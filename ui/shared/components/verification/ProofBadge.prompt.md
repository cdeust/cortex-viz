**ProofBadge** — the signature primitive: the verdict of a verification gate, in the brand's exact lexicon. Green `proven`, amber `unsourced`, red `blocked`, neutral `pending`.

```jsx
<ProofBadge status="proven" />
<ProofBadge status="unsourced">1 magic number</ProofBadge>
<ProofBadge status="blocked">2 violations</ProofBadge>
```

Use this for pass/fail of a check or a sourced/unsourced claim. For a node's *type* use `Badge`.

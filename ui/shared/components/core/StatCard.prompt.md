**StatCard** — a KPI tile for the metric strip: large mono value, uppercase micro-label, optional delta. Compose 3–5 in a row.

```jsx
<StatCard label="Memories" value="87" accent />
<StatCard label="Avg heat" value="0.342" />
<StatCard label="R@10" value="98.4%" delta="+33.4% vs flat" deltaDir="up" />
```

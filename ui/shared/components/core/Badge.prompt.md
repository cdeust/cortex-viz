**Badge** — a small uppercase mono label for a node's type (SEMANTIC / EPISODIC / ENTITY) or a compact status. Outlined by default; five tones.

```jsx
<Badge tone="accent">Semantic</Badge>
<Badge tone="ok" dot>Verified</Badge>
<Badge tone="danger" dot>Blocked</Badge>
```

For the pass/fail/unsourced verdict of a gate, prefer `ProofBadge` (verification group) — it carries the exact zetetic lexicon.
